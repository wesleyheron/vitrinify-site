# Vitrinify (website)

Site desenvolvido para o Mega Hack 2020 - 2ª edição (Equipe 6) (Desafio VTEX)

https://www.megahack.com.br/

## Install the dependencies

```bash
yarn
```

### Start the website in development mode

```bash
yarn start
```

### Build the website for production

```bash
yarn build
```
